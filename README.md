# tissue-optical-clearing-knowledge-base
Supporting data tables for the tissue optical clearing knowledge base used in my master’s thesis.

# 组织光透明知识库配套数据

本仓库为硕士学位论文《基于组织光透明知识库的方案生成方法研究》的配套数据仓库，用于存放论文中组织光透明知识库相关的完整数据表、字段说明及补充材料。

## 仓库说明

由于论文篇幅限制，正文中仅展示了知识库数据表的结构说明和部分示例记录。为便于查阅、复现与后续扩展，知识库相关完整数据表整理后统一存放于本仓库中。

本仓库中的数据主要对应论文第二章“组织光透明知识库的设计与构建”以及附录中的相关内容。

## 数据内容

本仓库主要包含以下几类数据表：

- `toc_cases.csv`  
  光透明案例表。记录具体样本在特定光透明方法下的样本表征信息、实验设置信息及结果信息。

- `toc_methods.csv`  
  光透明方法摘要表。记录不同光透明方法的方法名称、方法性质及各时间档位下的总处理时长等方法级信息。

- `toc_template_steps.csv`  
  光透明方法步骤明细表。记录各方法标准流程中的步骤顺序、阶段名称、试剂组成、实验条件及时间占比等步骤级信息。

- `toc_pretreatments.csv`  
  光透明前处理信息表。主要记录脱钙等前处理步骤的试剂、条件及各档位对应的标准处理时长。

- `fluor_marker_methods.csv`  
  荧光标记方法表。记录常用荧光标记方式的名称、标记对象、类别、具体形式及荧光团信息。

- `fluor_compatibility.csv`  
  光透明方法—荧光兼容关系表。记录不同光透明方法与荧光标记方法之间的兼容关系。

若个别字段名称与论文中表述存在中英文或书写形式差异，以本仓库数据表字段名为准，论文中的中文释义作为辅助说明。

## 数据组织原则

本知识库面向组织光透明方案生成任务构建，采用案例层、方法层和荧光约束层的组织思路：

- **案例层**：记录样本表征信息、处理设置与实验结果  
- **方法层**：记录光透明方法摘要信息与流程模板  
- **荧光约束层**：记录荧光标记方法及其与光透明方法之间的兼容关系  

其中，案例表中的部分字段与方法表、步骤表及兼容关系表通过编号或关联字段建立对应关系，用于支持后续的约束筛选、结构决策、时间预测与方案展开。

## 使用说明

1. 本仓库数据主要用于支撑论文中的知识库展示、方法说明和结果复核。
2. 若用于后续研究复现或扩展，建议结合论文正文中的方法描述一并阅读。
3. 各表中的字段命名、档位设定及兼容关系判定方式均服务于组织光透明方案生成任务，使用时应注意其任务背景。
